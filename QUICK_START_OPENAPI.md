# Быстрый старт: Использование OpenAPI спецификации

## Что было сделано

OpenAPI спецификация теперь используется для автоматической генерации типобезопасного API клиента.

## Шаги для начала работы

### 1. Установите зависимости

```bash
npm install
```

### 2. Сгенерируйте TypeScript типы

```bash
npm run generate:api-types
```

Это создаст файл `lib/api-types.ts` с типами из вашей OpenAPI спецификации.

### 3. Используйте новый API клиент

Вместо старого `lib/api.ts` используйте новый `lib/api-client.ts`:

```typescript
import { newsApi } from '@/lib/api-client'

// Получить опубликованные новости
const news = await newsApi.getPublished()

// Получить новость по ID
const newsItem = await newsApi.getById(1)

// Получить топ новостей
const topNews = await newsApi.getTop(10)
```

## Преимущества

✅ **Типобезопасность** - TypeScript проверяет правильность использования API  
✅ **Автодополнение** - IDE подсказывает доступные методы и параметры  
✅ **Автоматическая синхронизация** - при обновлении API просто перегенерируйте типы  
✅ **Меньше ошибок** - ошибки типов обнаруживаются на этапе разработки  

## Обновление при изменении API

Когда бэкенд обновляет API:

1. Обновите файл `openapi.yaml`
2. Выполните `npm run generate:api-types`
3. TypeScript покажет все места, где нужно обновить код

# Новостной лендинг - Агентство сельскохозяйственного развития

Современный адаптивный лендинг с мультиязычностью для агентства сельскохозяйственного развития.

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - Типизация
- **TailwindCSS** - Стилизация
- **React Hooks** - Управление состоянием

## Архитектура

- **Фронтенд**: Next.js + TypeScript (этот репозиторий)
- **Бэкенд**: Python (отдельный репозиторий, разрабатывается другим разработчиком)

## Функциональность

✅ Шапка с логотипом и навигацией  
✅ Dropdown меню для разделов (Документы, Проекты, О агентстве)  
✅ Мультиязычность (UZ / RU / EN)  
✅ Новостная секция с горизонтальной каруселью  
✅ Hero секция с анимациями  
✅ Секция "Основные направления"  
✅ Форма обратной связи (интеграция с API)  
✅ Плавающие кнопки (звук, чат, прокрутка наверх)  
✅ Полная адаптивность (мобилка + планшет + десктоп)  
✅ Зеленая цветовая гамма (агротематика)  
✅ API клиент для интеграции с Python бэкендом

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Создайте файл .env.local с настройками API
# NEXT_PUBLIC_API_URL=http://localhost:8000/api
# YOUTUBE_API_KEY=ваш_youtube_api_ключ (опционально, для автоматической загрузки видео)

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Интеграция с бэкендом

Фронтенд готов к интеграции с Python бэкендом:

1. **API клиент** находится в `lib/api.ts`
2. **Документация API** для бэкенд разработчика: `API_DOCUMENTATION.md`
3. **Инструкции по интеграции**: `README_API.md`

Настройте переменную окружения `NEXT_PUBLIC_API_URL` для подключения к вашему API.

## Автоматическая загрузка видео с YouTube

Приложение поддерживает автоматическую загрузку видео с канала YouTube:

1. **Получите YouTube API ключ** (см. `YOUTUBE_API_SETUP.md`)
2. **Добавьте в `.env.local`**:
   ```env
   YOUTUBE_API_KEY=ваш_api_ключ
   ```
3. Видео будут автоматически загружаться с канала @Agrosanoat_uz

Подробная инструкция: `YOUTUBE_API_SETUP.md`

## Особенности реализации

- **Ранние возвраты** для читаемости кода
- **Tailwind классы** для всех стилей
- **Описательные имена** переменных и функций
- **Префикс `handle`** для обработчиков событий
- **Доступность (a11y)**: aria-label, tabindex, keyboard navigation
- **TypeScript типы** для всех компонентов
- **DRY принцип** - без повторений кода
